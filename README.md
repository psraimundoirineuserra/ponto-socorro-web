## Sobre o Projeto

Website institucional do Centro Eclético da Fluente Luz Universal - Ponto Socorro de Cura Raimundo Irineu Serra.

### Stack

- **Framework:** Astro 6.x
- **Estilização:** Tailwind CSS v4
- **Estado:** Nanostores
- **Idiomas:** PT / ES / EN

---

## Desenvolvimento

```bash
npm install
npm run dev
```

---

## Estrutura

```
src/
├── components/     # Componentes Astro/React
├── data/          # Dados mock (hinos, produtos)
├── i18n/          # Traduções e configuração de idiomas
├── layouts/       # Layouts principais
├── pages/         # Rotas da aplicação
└── stores/        # Estado global (cart, audio)
```
